The missing layer for agent fleets

Terminal AI agents store each chat and can resume it — but nothing tracks which sessions were open, in which panes, when the machine went down. Deck is that layer.

♻️

Grid restore

Rebuild the exact window → tab → pane layout after a reboot, resuming each session in its own directory.

🎯

No guessing

Each session stamps its own process id. Deck keeps a session only if that process is alive — so idle-but-open panes are never dropped.

💾

Transcript backup

Every conversation is mirrored to a safe folder — and kept even if the original is deleted. A wipe can't lose your work.

✅

Proof before you restart

deck status reconciles your live panes against what's recorded and shows a plain verdict — so you know you're safe.

How it works

Install once, then forget it. Deck runs quietly and rebuilds everything when you need it.

Install

deck install wires a lightweight recorder into Claude Code and a 1-minute snapshot.

Work

Open as many sessions as you like. Deck records each one and mirrors its transcript in the background.

Restart

Reboot for any reason. Run deck restore — your full grid comes back, exactly as it was.
